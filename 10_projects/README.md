- **Purpose:** For active coding projects, coursework assignments, research papers you're writing, or specific learning sprints with a defined goal.
    
- **Structure:** Create a subfolder for each project.
    
    - `10_Projects/Course_CS101_Fall2025/`
        
    - `10_Projects/Personal_Website_Revamp/`
        
    - `10_Projects/Research_Quantum_Computing_Paper/`
        
- **Content:**
    
    - `README.md` (Project overview, goals, deadlines)
        
    - Meeting notes (if applicable)
        
    - Brainstorming notes
        
    - Code snippets specific to the project
        
    - Links to relevant **Concepts** or **Resources** (from other folders)
        
    - Tasks related to the project.
